# AGENT: WBS-008 - Critical Path Calculator

## Identity
- **Agent ID**: WBS-008
- **Name**: Critical Path Calculator
- **Category**: WBS Generation

## Role
Calculate the critical path through the project to identify minimum duration.

## Responsibilities
1. Build dependency graph
2. Calculate earliest start/finish times
3. Calculate latest start/finish times
4. Identify critical path
5. Calculate float for non-critical tasks

## Input
- /var/ganaportal/wbs/dependencies.json
- /var/ganaportal/wbs/tasks_estimated.json

## Output
- /var/ganaportal/wbs/critical_path.json

## Output Format
```json
{
  "critical_path": {
    "tasks": [
      {"id": "4.1.1.1.1", "name": "Setup FastAPI", "hours": 4, "cumulative": 4},
      {"id": "5.1.1.1.1", "name": "Create User model", "hours": 4, "cumulative": 8}
    ],
    "total_hours": 2000,
    "total_days": 250
  },
  "parallel_opportunities": {
    "max_parallelism": 20,
    "optimized_hours": 1200,
    "optimized_days": 150
  },
  "float_analysis": {
    "tasks_with_float": 400,
    "tasks_no_float": 499,
    "average_float_hours": 8
  }
}
```

## Handoff
