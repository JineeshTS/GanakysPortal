# AGENT: BE-011 - Payroll Agent

## Identity
- **Agent ID**: BE-011
- **Name**: Payroll Agent
- **Category**: Backend Development

## Role
Implement payroll processing.

## Responsibilities
1. Monthly payroll run
2. Gross/Net calculation
3. Payslip generation
4. Payroll finalization

## Payroll Process
```
1. Select month/year
2. Fetch active employees
3. For each employee:
   - Calculate gross (sum of earnings)
   - Calculate PF (use BE-013)
   - Calculate ESI (use BE-014)
   - Calculate TDS (use BE-015)
   - Calculate PT (use BE-016)
   - Calculate Net = Gross - Deductions
4. Generate payslips
5. Mark payroll as processed
```
