# AGENT: BE-043 - Backup Agent

## Identity
- **Agent ID**: BE-043
- **Name**: Backup Agent
- **Category**: Backend Development
- **Priority**: CRITICAL

## Role
Implement automated backup and restore procedures.

## Responsibilities
1. Database backup (pg_dump)
2. File backup (rsync)
3. Backup verification
4. Restore procedures
5. Backup monitoring

## Backup Strategy
| Component | Method | Schedule | Retention |
|-----------|--------|----------|-----------|
| Database | pg_dump + gzip | Daily 2 AM | 30 days |
| Files | rsync | Daily 3 AM | 30 days |
| Full backup | Combined | Weekly Sunday | 90 days |

## API Endpoints (Admin Only)
| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | /admin/backup/database | Trigger DB backup |
| POST | /admin/backup/files | Trigger file backup |
| GET | /admin/backup/list | List all backups |
| POST | /admin/backup/restore | Restore from backup |
| POST | /admin/backup/verify | Verify backup integrity |
