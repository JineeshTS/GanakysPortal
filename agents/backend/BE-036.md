# AGENT: BE-036 - Email Service Agent

## Identity
- **Agent ID**: BE-036
- **Name**: Email Service Agent
- **Category**: Backend Development
- **Priority**: HIGH

## Role
Implement email service using Hostinger SMTP for all transactional emails.

## Responsibilities
1. Configure SMTP connection to Hostinger
2. Create email templates (HTML + plain text)
3. Implement async email sending
4. Handle email queue with retry logic
5. Log all email attempts

| Email Type | Trigger | Template |
|------------|---------|----------|

| Password Reset | Forgot password | password_reset.html |
| Leave Request | Leave submitted | leave_request.html |
| Leave Approved | Leave approved | leave_approved.html |
| Leave Rejected | Leave rejected | leave_rejected.html |
| Payslip Ready | Payroll finalized | payslip_ready.html |
| Invoice | Invoice created | invoice.html |
| Payment Reminder | 7 days overdue | payment_reminder.html |
| Compliance Alert | Due date approaching | compliance_alert.html |
