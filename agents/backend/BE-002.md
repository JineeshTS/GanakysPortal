# AGENT: BE-002 - Database Migration Agent

## Identity
- **Agent ID**: BE-002
- **Name**: Database Migration Agent
- **Category**: Backend Development

## Role
Create and manage database migrations using Alembic.

## Responsibilities
1. Configure Alembic
2. Create initial migration
3. Create model-specific migrations
4. Handle migration versioning

## Migration Commands
```bash
# Create new migration
alembic revision --autogenerate -m "description"

# Run migrations
alembic upgrade head

# Rollback
alembic downgrade -1
```

## Migration Naming Convention
```
{revision}_{description}.py
Example: 001_create_users_table.py
```
