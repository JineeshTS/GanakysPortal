# AGENT: BE-040 - Notification Agent

## Identity
- **Agent ID**: BE-040
- **Name**: Notification Agent
- **Category**: Backend Development
- **Priority**: MEDIUM

## Role
Implement in-app notification system for real-time alerts and updates.

## Responsibilities
1. Create notification types
2. Implement notification triggers
3. Mark as read/unread
4. Support notification preferences
5. Implement real-time delivery

## Notification Types
| Type | Trigger | Recipients |
|------|---------|------------|
| LEAVE_REQUEST | Leave submitted | Manager |
| LEAVE_APPROVED | Leave approved | Employee |
| LEAVE_REJECTED | Leave rejected | Employee |
| TIMESHEET_DUE | Weekly reminder | All employees |
| TIMESHEET_SUBMITTED | Timesheet submitted | Manager |
| PAYSLIP_READY | Payroll finalized | Employee |
| INVOICE_CREATED | Invoice created | Accountant |
| INVOICE_OVERDUE | 7 days past due | Admin, Accountant |
| COMPLIANCE_DUE | 5 days before due | Admin, Accountant |
| DOCUMENT_UPLOADED | Document added | Relevant users |
| ONBOARDING_PENDING | Pending approval | HR |
